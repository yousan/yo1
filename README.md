# hubot-yo1

A hubot script that does the things

See [`src/yo1.coffee`](src/yo1.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-yo1 --save`

Then add **hubot-yo1** to your `external-scripts.json`:

```json
[
  "hubot-yo1"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
